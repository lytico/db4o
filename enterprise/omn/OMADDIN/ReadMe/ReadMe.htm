Welcome to ObjectManager Enterprise v1.0 for Visual Studio .NET

 

 

The Product: ObjectManager Enterprise for .NET from db4objects is a new, feature-rich product that aims to give db4o application developers a completely new experience that will enhance their productivity and help in delivering Persistence solution in still lesser time.

 

Presently available for the Windows environment as a Visual Studio .NET 2005 add-in, it lets developer perform all db4o activities from within the IDE, thus eliminating the need to deal with multiple applications or windows for browsing and querying a db4o database or checking status of Support Cases. Read on for detailed description of these and other features.

 

 

Features: Following are some of the highlight features of ObjectManager Enterprise:

 

·         Ad-hoc Querying: Allows the developer to connect and execute ad-hoc queries on any db4o database. User can browse classes and select, drag and drop attributes for querying. He can choose which attributes should be seen in the results which can be seen in a tabular as well as tree structure. User can edit attribute values for objects and save them back to the database.

 

·         Access to Support Cases: User can create new support cases as part of dDN support and expect a guaranteed reply within 24 hours. They can then follow-up on these cases. All this without leaving the Visual Studio .NET IDE.

 

·         Access to XtremeConnect: For more complex support requirements, licensed users can seek help from a db4o support engineer in a pairing session. This feature will allow users to share their desktop/application with support engineers and then jointly walk/debug through the application to communicate and resolve issues in an interactive, real-time environment.

 

·         Database Maintenance: Users can now carry out routine database maintenance such as Backing Up of db4o database at the click of a button.

 

 

 

Platform Requirements

·         Windows XP or Windows Vista

·         Visual Studio .NET 2005

·         db4o for .NET 6.3
